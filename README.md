# 🎨 Swyss - Developer Toolkit

Uma aplicação web minimalista e modular que centraliza suas ferramentas de desenvolvimento em um único dashboard personalizável. Um Canivete para Dev.

## ✨ Features

- **Grid Responsivo**: Organize seus blocos com drag-and-drop
- **Gerador de Paletas**: Crie paletas de cores harmoniosas com lock de cores
- **To-Do List**: Organize suas tarefas com uma interface limpa
- **Cronômetro & Temporizador**: Rastreie seu tempo de trabalho
- **Totalmente Minimalista**: Design dark, sem distrações

## 🚀 Tech Stack

- **React** + **Vite** (fast build)
- **react-grid-layout** (drag-and-drop)
- **Colormind API** (gerador de cores)
- **CSS puro** (sem frameworks)
- **Arch Linux + KDE**

## 📝 Componentes

- **GridComponent**: Gerencia o layout e blocos
- **ToDoList**: Gerenciador de tarefas
- **PaletteGenerator**: Gerador de paletas com Colormind API
- **TimeBlock**: Cronômetro e temporizador integrados

## 💡 Filosofia

Swyss é uma canivete para desenvolvedor com uma estrutura expansiva.
